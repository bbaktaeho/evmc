# BOOTSTRAP.md

Quick reference hub for the evmc project. Each section gives a one-line summary; follow the link for full details.

## Reading Order

처음 이 프로젝트를 접한다면 아래 순서로 읽는 것을 권장합니다:

1. **Architecture** – 프로젝트 구조와 핵심 설계를 파악합니다
2. **Commands** – 빌드, 테스트, 린트 명령어를 확인합니다
3. **Development Guide** – RPC spec 확인 → 워크플로우 → 코드 컨벤션 순서로 개발합니다
4. **Testing** – 테스트 작성 패턴과 주의사항을 확인합니다
5. **Known Limitations** – 현재 미구현 항목과 금지 사항을 인지합니다
